# Netflix Data Analysis

## Overview
This project provides an exploratory data analysis (EDA) of a comprehensive Netflix dataset. The analysis covers content trends over time, genre popularity, content ratings distribution, top directors, and geographical content distribution on the platform.

## Files Included
- `netflix_analysis.ipynb`: Jupyter Notebook containing the full step-by-step analysis, data cleaning, and visualization.
- `Netflix_Report.pdf`: A formatted PDF report summarizing the key findings and insights.
- `README.md`: This file explaining the project and how to use it.

## Key Insights
- Monthly trends showing when Netflix adds the most shows and movies, indicating strategic content release timings.
- Breakdown of content ratings, highlighting Netflixâ€™s focus on family-friendly and widely accessible content.
- Comparison between the volume of TV shows and movies available on Netflix.
- Identification of the most prolific directors contributing to Netflix's content library.
- Analysis of the most common genres on Netflix, revealing audience preferences and content focus.
- Geographic distribution of content, emphasizing dominant countries producing Netflix titles.
- Visualizations including bar charts, line charts, and pie charts for a comprehensive understanding.
- Data cleaning and preprocessing steps including handling missing values and data type conversions.

## Technologies Used
- Python 3.x
- Pandas for data manipulation
- Matplotlib and Seaborn for data visualization
- Jupyter Notebook for analysis and reporting
